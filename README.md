# bilemo
#API Bilemo

**Prerequisites :**

*   Php 7.1 or higher
*   Composer 1.8.6 or higher

**Installation :**

*   Clone the git.
*   Go to the folder bilemo.
*   Use this command : "composer install"
*   Configure database access in .env file.
*   Import "bilemo.sql" in your database.
*   Ready to use! enjoy ;)
