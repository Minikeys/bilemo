# API Bilemo

## Prerequisites :

*   Php 7.1 or higher
*   Composer 1.8.6 or higher

## Installation :

1.  Clone the git.
2.  Go to the folder bilemo.
3.  Use this command : "composer install"
4.  Configure database access in .env file.
5.  Import "bilemo.sql" in your database.
6.  Ready to use! enjoy ;)
